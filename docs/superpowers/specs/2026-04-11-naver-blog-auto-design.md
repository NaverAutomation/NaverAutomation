# 네이버 블로그 자동화 솔루션 (npx-naver-auto) 설계 문서

## 1. 개요
사용자가 터미널에서 `npx` 명령어로 간편하게 실행할 수 있는 로컬 웹 기반 네이버 블로그 자동 포스팅 프로그램입니다. GPT를 활용한 원고 생성, DALL-E 기반 이미지 생성, 그리고 Playwright를 이용한 안전한 브라우저 자동화를 특징으로 합니다.

## 2. 핵심 목표
- **사용자 편의성:** 설치 과정 없이 `npx` 명령어로 즉시 실행.
- **안전한 자동화:** 사용자의 로컬 IP와 실제 브라우저 세션을 활용하여 네이버의 봇 감지 회피.
- **AI 기반 콘텐츠 생성:** 키워드만으로 고품질의 블로그 원고와 이미지를 자동 생성 및 수정.
- **다중 계정 및 예약 관리:** 여러 네이버 계정을 순환하며 포스팅하고 정교한 예약 스케줄링 지원.

## 3. 기술 스택
- **Runtime:** Node.js
- **Backend:** Express.js, SQLite3 (로컬 데이터 저장), Socket.io (실시간 로그)
- **Frontend:** React, Tailwind CSS, Lucide React (아이콘)
- **Automation:** Playwright (Chromium 제어)
- **AI Integration:** OpenAI API (GPT-4o, DALL-E 3)
- **Security:** AES-256 (민감 정보 암호화)

## 4. 상세 기능 설계

### 4.1. 실행 및 UI 구조 (npx 환경)
- 사용자가 `npx naver-auto` 입력 시 로컬 서버(Port 3000 등) 기동.
- 시스템 기본 브라우저를 통해 대시보드(`http://localhost:3000`) 자동 오픈.

### 4.2. 계정 및 설정 관리
- **다중 계정 지원:** 네이버 ID/PW를 여러 개 등록하고 상태(정상, 제한 등) 관리.
- **API 설정:** OpenAI API Key 입력 및 로컬 암호화 저장.
- **데이터 로컬 보관:** 모든 설정과 계정 정보는 사용자 컴퓨터 내 SQLite DB 파일에 저장.

### 4.3. 콘텐츠 자동 생성기
- **키워드 기반 원고 생성:** 주제 입력 시 GPT가 제목 및 본문(Markdown/HTML) 생성.
- **이미지 생성/관리:** DALL-E 3를 이용한 이미지 생성 또는 로컬 사진 업로드 지원.
- **원고 수정 및 최적화:** 생성된 글을 사용자가 직접 수정하거나 GPT에게 재요청(프롬프트 피드백).

### 4.4. 예약 및 포스팅 엔진
- **Playwright 자동화:** 
    - 실제 브라우저를 띄워 네이버 로그인 수행.
    - 스마트 에디터 ONE 환경에서 제목, 본문, 이미지 삽입 및 서식 적용.
    - 발행 버튼 클릭 자동화.
- **스케줄러:** 
    - 네이버 자체 예약 기능 활용 옵션.
    - 프로그램 자체 스케줄러를 통한 시간차 포스팅(다중 계정 순환 발행).
- **실시간 모니터링:** 포스팅 과정을 웹 UI 로그 창에 실시간으로 출력.

## 5. 보안 및 데이터 정책
- **비밀번호 보호:** 네이버 비밀번호 및 API Key는 AES-256 알고리즘으로 암호화하여 DB 저장.
- **IP 정책:** 서버 호스팅이 아닌 로컬 실행 방식이므로 사용자의 실제 접속 IP를 유지하여 저품질 방지.

## 6. 예외 처리
- **로그인 차단:** 캡차(CAPTCHA) 발생 시 사용자에게 브라우저 직접 조작 요청(알림).
- **네이버 UI 변경:** 스마트 에디터 구조 변경 시 유연하게 대응 가능한 선택자(Selector) 관리 로직 적용.
